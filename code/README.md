## Reproducibility

A typical workflow is:
1. Run `download_data.R` to obtain and prepare the earthquake data.
2. Run `generate_maps.R` and `analyze_hotspots.R` for spatial exploration.
3. Run `time_domain_analysis.R` and `spectral_analysis.R` for temporal diagnostics.
4. Run `modeling_and_forecasting.R` for ARIMA/ETS forecasts.
